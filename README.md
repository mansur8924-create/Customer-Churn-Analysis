# Customer-Churn-Analysis
 
 Project Overview

The objective of this project is to identify the key factors that lead to customer turnover (churn) in a telecommunications company. By analyzing customer behavior, contract types, and demographics, I developed a data-driven approach to predict which customers are most likely to leave, allowing the business to take proactive retention measures.

 Tools & Technologies

Python: Used for the entire end-to-end data pipeline.

Pandas & NumPy: Used for data cleaning and manipulation.

Matplotlib & Seaborn: Used to create professional visualizations.

Scikit-Learn: Used for building and training the machine learning model.

 Project Structure

Data Cleaning: Removing null values and preparing the dataset for analysis.

Exploratory Data Analysis (EDA): Visualizing churn patterns based on contract type and tenure.

Model Training: Using a classification model to predict future churn.

 Key Insights

Contract Impact: Customers on "Month-to-Month" contracts have a significantly higher churn rate compared to those on long-term contracts.

Tenure: New customers are at the highest risk of leaving within the first 6 months of service.

Visualizations: Detailed charts were generated to show the distribution of churn across different customer segments.

 How to Use

Clone this repository.

Ensure you have the Telco_Churn_Cleaned.csv file in the same directory as the scripts.

Run 2_data_cleaning.py to see the visualization generation process.
